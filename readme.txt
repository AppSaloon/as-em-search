== Description ==

This plugin enables an admin to search bookings made through use of the Events Manager plugin.
It will search in every field tied to a booking for the entered value and show the results.
To access the search form, click on Search in the Events Manager menu.

== Installation ==
There are two ways to obtain the plugin, either download it as a zip file from Github or clone the repository.

If you downloaded the plugin as a zip file
1. Unzip the download file
2. Move the as-em-search folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

To clone the repository from Github, make sure you are in the `/wp-content/plugins/` directory when you issue the clone
command.
This way it's in the correct folder and you can update the plugin by going to the `/wp-content/plugins/as-em-search`
directory and use git pull to obtain the latest version.

Don't forget to activate the plugin through the 'Plugins' menu in WordPress
